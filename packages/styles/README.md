# @arco/styles

用于多个 `app` 公用的样式文件，继承了 `@arco-core/design` 的所有能力。业务上有通用的样式文件可以放在这里。

## 用法

### 添加依赖

```bash
# 进入目标应用目录，例如 apps/xxxx-app
# cd apps/xxxx-app
pnpm add @arco/styles
```

### 使用

```ts
import '@arco/styles';
```
